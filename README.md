# DubiDero

DubiDero es una herramienta de entrenamiento para League of Legends enfocada en practica, clears, rutas de jungla y seguimiento de progreso personal.

## Descargar DubiDero

La version publica para Windows estara disponible desde la seccion de releases:

[Descargar DubiDero](https://github.com/DubiDero/DubiDero/releases/latest)

El instalador descarga la version mas reciente de la app y la mantiene preparada para futuras actualizaciones.

## Caracteristicas

- Practica de entrenamiento con bots.
- Soporte para rutas y clears.
- Records personales y oficiales cuando inicias sesion.
- Practica offline sin marcar records oficiales.
- Actualizaciones integradas.

## Estado del proyecto

DubiDero esta en desarrollo activo. Algunas funciones pueden cambiar mientras se estabilizan las primeras versiones publicas.
